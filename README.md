# MomentumCloneCoding
- 크롬 확장 프로그램 Momentum의 클론코딩 입니다 

## 기능
1. 실시간 시계, 현재 지역 날씨 정보
2. 사용자 이름 등록
3. 할 일 목록 생성 및 삭제
4. 사용자 이름, 할 일 목록 자동으로 스토리지에 저장 및 불러오기
5. 페이지를 시작할 때 배경화면 8개중에 랜덤으로 선택하여 적용
